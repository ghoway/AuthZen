# **App Name**: AuthZen

## Core Features:

- User Registration: Registration page with fields for full name, email, password, confirm password, and acceptance of Terms and Conditions.
- User Login: Login page using email and password credentials.
- User Dashboard: Dashboard displaying the logged-in user's name and email in a blurred card.
- Database Integration: Integration with Prisma ORM to manage user data, including authentication details, roles, and relationships.

## Style Guidelines:

- Background: A gradient from dark purplish gray (#1c1a2e) to dark purplish red (#8f5c5c), with a transition through a desaturated purple (#3a2d54) to provide a dark and engaging backdrop.
- Accent: Yellow gold (#f3c222) inspired by chameleon eyes and scales for a standout element.
- Call-to-action buttons: Indigo blue (#4f46e5) for high contrast and clear visibility.
- Text: White and light grays for easy readability against the dark background.
- Font pairing: 'Inter' (sans-serif) for both headlines and body text to give a modern and readable aesthetic.
- Cards with a blurred background and white borders, creating a frosted glass effect for content areas on the login and dashboard pages.
